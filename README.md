# Multi-Domain Data Analysis & Visualization Portfolio – Month 2

## Overview
This repository contains my **Month 2 Data Analysis & Visualization Portfolio**, consisting of 5 distinct projects across multiple domains:
- Retail (Supermarket Sales)
- Education (Student Performance)
- Weather (Climate Data Analysis)
- Healthcare (COVID Trends)
- Finance (Stock Market Analysis)

Each project demonstrates data cleaning, transformation, visualization, and generation of actionable insights using Python and pandas.
